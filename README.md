# User Management Dashboard

A React-based user management application built with Vite and using the JSONPlaceholder REST API.

## Features

- View users from API
- Add new users
- Edit existing users
- Delete users
- Search users
- Sort users (A-Z / Z-A)
- Pagination options:
  - 10
  - 25
  - 50
  - 100
- Filter users
- Responsive interface
- Client-side form validation
- API error handling
- Unit testing


## Tech Stack

- React
- Vite
- Axios
- Jest
- React Testing Library
